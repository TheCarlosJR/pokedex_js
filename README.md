# Pokedex JS

Aplicativo web desenvolvido em javascript para servir de interface da <a href="https://pokeapi.co/" target="_blank">`PokeAPI`</a> a fim de criar uma Pokedex.
Pokedex é um catálogo de monstros fictícios da franquia "Pokemon".

Neste caso, a **Pokedex JS** é totalmente **interativa** e **adaptável** aos mais diversos dispositivos de acesso tais como smarphones, tablets e computadores em geral.

<img src="https://github.com/TheCarlosJR/pokedex_js/blob/main/prints/0.png" title="Desktop" width="580" height="580" /> <img src="https://github.com/TheCarlosJR/pokedex_js/blob/main/prints/1.png" title="Smartphone" height="580" />

O projeto foi desenvolvido com o objetivo de cumprir um desafio do `"Santander Bootcamp 2023 - Fullstack Java+Angular"`.
